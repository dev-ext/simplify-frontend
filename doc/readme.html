Simplify Bolier Plate Documentation

Simplify Bolierplate created with Twitter Bootstrap and inspired by html5boliderplate for frontend development. It is template like html5boliderplate

Asset description

- fonts Store non system font files.
-- selection.json This Bolierplate have own customize icon set. This icon set generated with icomoon.io. For further modification and customozation this file need to be upload to icomoon.io, I give this icon set name to iconic.
- images Store all images here
- js Store all js files to js folder. Included bootstrap.js, jquery.fittext.min.js, jquery.fitvids.min.js,jquery.min.js,modernizer.min.js, respond.js and main.js and head.js included in main.js
-- lib All library and Plugin
- scss store all scss files. bootstrap scss included and other scss added as module structure. Also some wordpress default class added to this repo. Default it is disabled. uncomment line no 23 in style.scss file.
- config.rb It is required for compass app settings.
- favicon.ico This repository is for inspiration of html5boliderplate. For this reason I use html5boliderplate favicon.
- style.css Main and one stylesheet. This file will automatically generate. No need to edit this file manually if you use SCSS.
- style.scss Main scss file.
-.gitignore For who Use Git.
